# Casa
Exemplo de código de uma casa e um humanóide feitos com opengl
Para abrir o projeto, abra o arquivo Project1.fpj em um IDE que suporta a linguagem c++. Exemplo Falcon c++

Controles:
- Casa:
  W - Abre/Fecha aporta
  S - Abre/Fecha janelas
- Humanoide:
  Z - Anda para frente
  X - Vira 180º
